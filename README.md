# PyFinance-Tracker
# 💰 Sistema de Controle Financeiro em Python

Este projeto é um sistema simples de controle financeiro desenvolvido em Python, com execução via terminal. Ele permite registrar, visualizar e analisar transações financeiras de forma prática.

---

## 🚀 Funcionalidades

* ➕ Adicionar transações (entrada e saída)
* 📋 Listar todas as transações
* 💰 Calcular saldo atual
* 🔎 Filtrar transações por categoria
* 💾 Salvar e carregar dados automaticamente (JSON)

---

## 🧠 Conceitos aplicados

* Estruturas de dados (listas e dicionários)
* Funções em Python
* Condicionais e loops
* Manipulação de arquivos com JSON
* Validação de entrada e tratamento de erros
